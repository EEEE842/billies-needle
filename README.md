# Billies-Needle
-
Billies Needle is a universal "mod" injector for scratch and turbowarp. All you have to do is get the bookmarklet from the website then you can inject the menu! The menu includes 3 catagories as of writing this; misc, variables, and sprites. Each has many options to mod your game!
-
Jimmies Needle is an old "prototype" version of this, it claimed to inject into any game, it did, but poorly and cloud variables didn't work. Billies Needle is more advanced and works with cloud.
